# Research Watch: OpenAI Codex plugin system — Skills + Apps + MCP bundles

- Announcement: <https://x.com/openaidevs/status/2037296316104282119>

## Why this is worth watching
OpenAI has launched a **plugin system for Codex** that packages Skills (workflow prompts), Apps (tool integrations), and MCP servers into installable, team-shareable bundles. Official plugins now exist for GitHub, Linear, Vercel, Netlify, Hugging Face. This is a direct structural parallel to Claude Code's plugin/skill ecosystem — but from the second-largest AI lab.

## What stands out immediately
- **Bundle model:** one plugin = Skills + Apps + MCP servers together (not separately managed)
- **Team sharing:** plugins are reusable across team members — shifts from individual to team-level capability units
- **Day-1 integrations:** GitHub, Linear, Vercel, Netlify, Hugging Face, Slack, Figma, Notion, Gmail
- **Scope expansion:** Codex moves from code generation to full development workflow (planning, research, coordination)
- **Structural parallel to Claude Code plugins** — two major ecosystems converging on the same architecture

## Why clawfit should care
This validates that the **plugin bundle pattern** (Skills + MCP + tool integration in one installable unit) is becoming the standard at the platform level, not just community projects. clawfit's Level 4b (skill packs) taxonomy needs to distinguish between:
- Community skill packs (Impeccable, K-Skill, oh-my-* style)
- Platform-native plugin systems (Codex plugins, Claude plugins-official)
- Cross-platform managers (Chops — manages both)

The Chops skill manager (already tracked) becomes more strategically important as managing plugins across Claude Code and Codex simultaneously is now a real use case.

## Preliminary interpretation
- Primary: **Level 4b — Skill packs & skill managers** (platform-native plugin system)
- Secondary: **Level 4c — Tool-use / action infrastructure** (MCP bundles within plugins)

## Status
- High relevance. Competitor signal that reinforces clawfit's Level 4 taxonomy direction.
- Chops watch should be updated to note Codex plugin support.
