## Why this is worth watching
Roo Code is a VS Code extension that frames itself as "a whole dev team of AI agents in your code editor" — multi-persona agent roles (architect, developer, reviewer) operating within a single IDE session. At 23,503 stars it is a direct high-star-count rival to Cline in the IDE coding agent slot. Unlike Cline (single-agent assistant), Roo Code's "dev team" metaphor implies role-based multi-agent orchestration inside the IDE — a distinct positioning that could split the VS Code agent market.

## What stands out immediately
- "Whole dev team" framing: multiple role-specific agents (not just one assistant)
- VS Code extension delivery — same distribution channel as Cline and Continue
- TypeScript primary language; likely integrates via VS Code's agent SDK patterns
- 23,503★ puts it in the same tier as Cline (59k) and Continue (32k) but with faster recent growth
