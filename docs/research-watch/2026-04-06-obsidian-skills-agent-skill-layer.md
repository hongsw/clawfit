# Research Watch: kepano/obsidian-skills — Agent Skills for Obsidian

- Repo/Link: https://github.com/kepano/obsidian-skills
- Source: GitHub Trending

## Why this is worth watching
Kepano (Steph Ango, Obsidian CEO) shipping an "agent skills" layer that teaches agents to use Obsidian's Markdown, Bases, JSON Canvas, and CLI is a strong signal that L4b skill-pack patterns are maturing beyond coding environments. At 20k stars, it represents mainstream adoption of the idea that agents need taught, installable skills rather than hardcoded capabilities.

## What stands out immediately
- 20,148 stars with 281 added today — broad organic adoption
- Authored by the Obsidian CEO — first-party, ecosystem-sanctioned skill pack
- Covers Markdown, Bases (Obsidian's new database layer), JSON Canvas, and CLI
- Explicitly framed as "agent skills" — pure L4b pattern
- Positions Obsidian as an agent-native knowledge environment

## Why clawfit should care
Obsidian has become a popular personal knowledge base and note-taking tool for developers, researchers, and PMs — all roles clawfit profiles. A skill pack that makes agents fluent in Obsidfit workflows signals that L4b tooling is expanding from pure coding contexts into knowledge-work contexts. Clawfit's skill layer scoring should account for domain-specific skill packs beyond code-gen.
