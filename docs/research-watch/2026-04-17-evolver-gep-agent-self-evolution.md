## Why this is worth watching
Evolver applies Gene Expression Programming (GEP) to evolve AI agent behavior — using evolutionary algorithms rather than gradient descent or prompting to improve agent performance over time. This is a distinct approach from both RL-tuned agents and self-modifying agents like GenericAgent. JavaScript-first; positions itself as a "self-evolution engine" that can be attached to existing agents.

## What stands out immediately
