# Research Watch: obra/superpowers — Agentic Skills Framework & Methodology

- Repo/Link: https://github.com/obra/superpowers
- Source: GitHub Trending

## Why this is worth watching
superpowers is simultaneously an agentic skills framework and a software development methodology, with 145,706 stars and +2,150 today — one of the highest-starred repositories on GitHub Trending. At that scale it crosses from "project" into "de facto standard." The framing "that works" signals explicit positioning as a reliability-over-novelty framework, directly addressing the harness reliability problem the ecosystem has been circling.

## What stands out immediately
- Combined methodology + framework: not just a skills pack but a full development philosophy
- Shell-first implementation — portable, no dependency on any particular agent runtime
- 145k stars makes it the largest skills/harness repo in this taxonomy by a substantial margin
- The name "superpowers" echoes the skill-augmentation pattern (Level 4b) but the scope suggests Level 3 (team SSOT)
- Shell language choice implies broad compatibility: usable with Claude Code, OpenCode, Codex, Goose, Aider

## Why clawfit should care
obra/superpowers is the first Level 3 entry with enough adoption to influence org-wide conventions. Its star count exceeds every other harness/SSOT entry in the registry. If clawfit's recommendation engine is suggesting harness/SSOT tooling for mid-to-large teams, superpowers must appear in that shortlist. It also challenges the current boundary between Level 3 (team harness) and Level 4b (skill packs): superpowers spans both.
