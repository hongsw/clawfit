## Why this is worth watching
A domain-specialized Claude Code harness with 49 AI agents, 72 workflow skills, and a coordination system modeled on real game studio hierarchy. This is the most elaborately structured domain-specific multi-agent harness observed in this taxonomy — and its framing as a "studio" rather than a "tool" is architecturally significant.

## What stands out immediately
- 49 agents organized into studio roles (writers, designers, engineers, producers, testers)
- 72 skills covering full game dev pipeline (design, code, asset, QA, publishing)
- Coordination system explicitly mirrors professional studio org chart
- Shell-based like superpowers; requires only Claude Code
- 10k+ stars in early life signals strong developer-to-creator audience crossover

## Why clawfit should care
This is the clearest evidence that Level 2/3 domain harnesses are now fragmenting into industry-specific verticals. Previously, domain specialization appeared in Level 4b skill packs (obsidian-skills, K-Skill). Claude-Code-Game-Studios is the first high-signal Level 2/3 *harness* specialized for a non-software-engineering domain. It suggests clawfit may need a `domain` dimension alongside `task` in the scoring model to distinguish cross-domain tools from domain-specialized ones.
