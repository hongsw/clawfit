## Why this is worth watching
Vercel is the dominant deployment platform for frontend developers, and this is their official open-source template for building cloud-hosted agents. Platform vendors entering the agent template space directly shapes what "default" agent architecture looks like for the next generation of projects.

## What stands out immediately
- Official Vercel Labs repo — not a community fork
- TypeScript-based cloud agent template with clear production orientation
- Significant single-day star velocity (+915 in one day) suggests strong developer interest
- Positions cloud-hosted agents as the natural next step after deploying web apps on Vercel

## Why clawfit should care
Vercel's entry into cloud agent templates expands the Level 1 surface: developers who already live in the Vercel ecosystem now have a first-party template that bundles agent deployment with their existing hosting workflow. This may lower the activation energy for "online" + "solo/small team" profiles considerably — and positions cloud-hosted agent deployment as a commodity rather than an infrastructure challenge.

## Preliminary interpretation
Current best reading:
- **Level 1 — Base runtime / cloud agent template**
- Overlaps with Level 2 (harness/orchestration) if the template includes workflow configuration
