# YC / Show HN style intro draft

Title idea:
Show HN: clawfit — a layered map of the AI agent ecosystem

Body draft:
I’ve been building `clawfit`, a public map of the AI agent ecosystem.

The motivation was that most discussions still flatten everything into “AI coding tools,” even though the ecosystem is clearly splitting into layers:
- base runtimes
- harness / wrapper layers
- team workflow / executable SSOT systems
- capability extensions (MCP, memory, tools)
- research / benchmark patterns
- data / knowledge infrastructure
- human interface / voice layers
- methodology signals

So this repo is an attempt to make those structural differences visible.

Start here:
- Ecosystem map: https://github.com/hongsw/clawfit/blob/main/docs/reference-levels.md
- Intro: https://github.com/hongsw/clawfit/blob/main/docs/posts/2026-03-28-introducing-clawfit.md

- what’s missing
